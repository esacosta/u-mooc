Verifique su código

Como se mencionó en otras páginas, la creación de la página de inicio, lecciones, actividades y evaluaciones requiere la creación de archivos CSV (para la página principal y de estudio) y un conjunto de archivos JavaScript (para actividades y evaluaciones). Desarrollador del curso contiene una herramienta que permite escanear los archivos de referencias rotas y otros errores.

Tenga en cuenta que este verificador no puede capturar todos los errores. Es un primer paso, para encontrar ciertos problemas anticipados.Usted todavía debe probar su aplicación visitando las páginas del curso en el servidor de desarrollo local y comprobar que los enlaces y campos de entrada funcionan como se esperaba.

Ejecute la herramienta de verificación

Para ejecutar esta herramienta, abra una línea de comandos o el comando. En esa ventana:

cd AppPath 
python tools / verify.py

donde AppPath es el directorio que contiene la aplicación.

Lo que se verifica

La herramienta de verificación genera una lista de advertencias y errores que indican posibles datos no válidos en los archivos CSV y actividades incorrectamente formateados y evaluaciones. Por ejemplo, en los archivos CSV que las capturas numerado incorrectamente las unidades o elementos de una fila con demasiadas o muy pocas.
