Internacionalización

    • Traducciones de encargo

Curso Builder soporta 57 idiomas y todo estudiante que se traduce el texto en estos idiomas. Para habilitar esta funcionalidad ampliada idioma, siga los siguientes pasos:

  1. Descargue el paquete de idioma en el equipo.
  2. Descomprimir el archivo. Ahora tiene un directorio llamado i18n.
  3. Navegue en las modules / en la instalación de desarrollador de curso.
  4. Copie el directorio i18n nuevo que ha descomprimido en este directorio actual, sustituyendo completamente el directorio i18n antiguo por este nuevo directorio.

Para utilizar uno de estos idiomas, se especifica en el archivo course.yaml. Por ejemplo, para especificar la traducción al español, la línea local debe ser

locale: 'es'

El código de letras idioma a utilizar es el nombre del directorio de este idioma en el directorio local. La ruta completa al directorio del idioma es:

modules/i18n/resources/locale

Estructura del curso del constructor Internacionalización utiliza las siguientes tecnologías:

  • webapp2
  • pybabel
  • pytz
  • Extensión Jinja2 de i18n

Esta capacidad de internacionalización contó con la ayuda de UniMOOC-Tec ( http://www.unimooc-tec.com/ ) de la Universidad de Alicante.

Traducciones de encargo

Usted todavía puede hacer traducciones de idiomas personalizados fuera de los idiomas admitidos. Todas las cadenas de texto se colocan dentro de la {% trans%} {% endtrans %} comienzo y fin. Estas etiquetas están incrustadas dentro de las plantillas HTML, que están situados en los puntos de vista / directorio.

Por ejemplo, el código siguiente:

  {% Trans%} Unidad{% endtrans %}

indica que el texto "Unidad" se puede traducir.

Si cambia el texto por defecto dentro de estas etiquetas o añadir nuevas cadenas traducibles de su cuenta, usted debe actualizar el archivo. Po en

modules/i18n/resources/locale / <idioma> / LC_MESSAGES

A continuación, ejecute update pybabel.
