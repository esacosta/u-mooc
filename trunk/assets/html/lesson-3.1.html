Resumen

  1. Descargar Python 2.7.
  2. Descarga el Google App Engine SDK para Python.
  3. Descargue la aplicación de ejemplo de U-MOOC.

Descargar e instalar Python 2.7

Asegúrate de que tienes Python 2.7 en su sistema. 

Descargar e instalar el Google App Engine

Si no está familiarizado con GAE, pero sabe algo acerca de la programación en la nube, lea ¿Qué es Google App Engine? 

Descargue el SDK de GAE para Python.

  • PC: Ejecute el instalador MSI descargado haciendo doble clic. Siga todas las instrucciones que aparecen en pantalla
  • Mac: Abra el DMG descargado y arrastre el icono de GAE en la carpeta Aplicaciones. 
  • Linux u otro sistema compatible con POSIX: Después de descargar el archivo ZIP, vaya al directorio en el que desea instalar GAE. Al descomprimir el archivo, se crea un subdirectorio llamado google_appengine. Añadir el directorio a la ruta de su intérprete de comandos para que pueda encontrar las herramientas de línea de comandos cuando usted los necesite.

Descargar la aplicación de ejemplo de course-builder

Para descargar la aplicación del Generador de Curso, tiene dos opciones:

  • Descargue el archivo ZIP. Descomprimir el archivo en un directorio de trabajo. 
  • Visite la página de course-builder
    Desde allí, siga las instrucciones para obtener una copia local del repositorio usando Git. Utilice este método si está explícitamente asociado con Google para realizar cambios como desarrollador del curso
