En un punto anterior se ha visto que es necesario diseñar el curso a partir de una secuencia de objetivos, pero además de esto es esencial trabajar con cada uno  de estos objetivos por separado. Puede seguir una serie de pasos con cada uno de ellos (Ver ilustración 3):

Ilustración 3.- Pasos a seguir para cada objetivo [1]

1.       Haga una lista de cada conocimiento o habilidad que un estudiante necesita saber para completar el objetivo.

2.       Algunos de los componentes de conocimiento se pueden repetir durante la fase de planificación.

3.       Haga una lista de cada cosa que el estudiante debe ser capaz de hacer para completar el objetivo.

4.       Identificar las dependencias entre los componentes del conocimiento y las habilidades necesarias.

○        ¿Hay algo que debe saber antes de que puedan adquirir una habilidad?

○        ¿Hay ciertas habilidades que deben aprender primero para comprender mejor una habilidad posterior?

○        ¿Existen habilidades y conocimientos que son complementarios, es decir, aquellas que ayuden al alumno más fácilmente a aprender o adquirir competencias?

○        ¿Existen dependencias entre conocimientos diferentes?

○        ¿Existen dependencias entre habilidades?

5.       Reorganize la lista para que las dependencias se enumeren en orden.

6.       Revise los supuestos de la fase de planificación.

○        ¿Su lista contiene los conocimientos que el estudiante debe saber antes de comenzar el curso?

○        ¿Su lista contiene habilidades que un estudiante debe tener antes de comenzar el curso?

7.       Ahora debería tener una lista de piezas de conocimientos y habilidades que un estudiante necesita para completar con éxito su objetivo.

8.       Repita el procedimiento para cada objetivo.

9.       Una vez que haya completado los pasos de proceso por cada objetivo, compare las listas de cada objetivo y fíjese en lo siguiente:

○        Algunas habilidades y piezas de conocimiento puede ser comunes a través de diferentes objetivos.

○        Haga una verificación cruzada de dependencias entre los distintas habilidades y conocimiento de los objetivos y trate de organizarlos como elementos de una manera lógica.

10.   Combine las listas.

○        ¿Existen componentes de conocimiento que trascienden los diferentes objetivos?

○        ¿Existen habilidades complementarias que atraviesan diferentes objetivos?

○        ¿Existen otras dependencias en el aprendizaje de la materia y la adquisición de las habilidades?

○        Combinar elementos cuando sea necesario. Eliminar las piezas repetidas.
